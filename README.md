# Ozon API 客户端

Ozon广告API的Python客户端实现，用于管理广告活动、广告组和广告，以及获取相关统计数据。

## 功能特点

- 支持Ozon广告API的认证和请求签名
- 提供广告活动、广告组和广告的管理功能
- 支持获取广告效果统计数据
- 可扩展的API客户端设计

## 安装
pip install -r requirements.txt
## 使用方法

1. 配置环境变量或在配置文件中设置您的API凭据
2. 初始化客户端并调用相关方法
3. 查看示例代码了解更多用法
